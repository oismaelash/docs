Pilot Status lets you send rich media — images, videos, audio clips, documents, and stickers — through the same REST API you use for text messages. You provide either a publicly accessible URL or a base64-encoded string, and Pilot Status downloads and forwards the media to WhatsApp on your behalf. This guide covers each media type, supported formats, size limits, and how to attach captions to your media messages.

<Note>
  Media URLs must be publicly accessible. Pilot Status downloads and forwards them to WhatsApp — URLs behind authentication, VPNs, or private networks will fail to deliver.
</Note>

## Send a media message

Use the `/v1/messages/media` endpoint for all media types. Set the `type` field to match the kind of media you're sending, and provide the file via `url`.

<Tabs>

<Tab title="Image">

```bash curl
curl -X POST https://api.pilotstatus.com.br/v1/messages/media \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "number_id": "num_abc123",
    "to": "5511999999999",
    "type": "image",
    "url": "https://example.com/invoice.png",
    "caption": "Your invoice for January 📄"
  }'
```

</Tab>

```bash curl
  -d '{
```

```bash curl
  }'
```

Use the `filename` field to control the display name of the document in the WhatsApp chat. Without it, WhatsApp uses the last segment of the URL path.

## Supported formats and size limits

WhatsApp enforces strict limits on media type and file size. Requests that exceed these limits are rejected before delivery.

| Type | Supported formats | Max size |
|------|-------------------|----------|
| Image | JPG, PNG, WEBP | 5 MB |
| Video | MP4, 3GP | 16 MB |
| Audio | MP3, OGG, AAC | 16 MB |
| Document | PDF, DOCX, XLSX, PPTX | 100 MB |

<Warning>
  Animated GIFs are not natively supported as the `image` type. Convert them to MP4 and send as `type: "video"` instead, or send as a sticker (unofficial connections only — see below).
</Warning>

## Media with captions

The `caption` field lets you attach a text label beneath images, videos, and documents. Captions support plain text and emoji, and they appear inline in the chat beneath the media preview.

```bash curl
curl -X POST https://api.pilotstatus.com.br/v1/messages/media \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "number_id": "num_abc123",
    "to": "5511999999999",
    "type": "image",
    "url": "https://example.com/promo-banner.jpg",
    "caption": "🎉 Our biggest sale of the year — up to 50% off. Shop now!"
  }'
```

Keep captions concise. WhatsApp truncates very long captions in the notification preview, though the full text is visible when the chat is opened.

<Tip>
  For unofficial (QR-code) connections, you can also send stickers by setting `type: "sticker"` and providing a `.webp` file URL. Stickers are not supported on the official Cloud API integration.
</Tip>
