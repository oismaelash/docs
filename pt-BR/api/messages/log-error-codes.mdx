# Códigos de Erro do Log

Em **Logs** (`/logs`), o **Código de erro** exibido nos detalhes de um envio com falha é sempre um dos valores abaixo. A API do painel expõe o mesmo valor como **`errorCode`** em `GET /api/logs` (derivado no servidor a partir de campos armazenados; não é o mesmo que `errorMessage`).

## Códigos (conjunto fechado)

| Code | Significado |
| --- | --- |
| `DELIVERED_NOT_CONFIRMED` | A mensagem foi **enviada com sucesso**, mas a confirmação de entrega não chegou antes do prazo (**deliverUntil**); marcada como **FAILED** após o prazo. |
| `TEMPLATE_VERSION_NOT_FOUND` | Template ou versão do template não encontrado para a mensagem. |
| `INVALID_DESTINATION` | Destino inválido ou falha na normalização E.164. |
| `PILOT_SELF_NUMBER` | Envio bloqueado para o próprio número da Pilot Status na instância da Pilot Status. |
| `NEVER_ATTEMPTED_MAX_AGE_EXCEEDED` | A mensagem permaneceu **QUEUED** por 7 dias com zero tentativas de envio (a instância ficou offline o tempo todo); expirou após as verificações automáticas de prazo. |
| `EXPIRED_AFTER_ATTEMPTS` | **deliverUntil** expirou após pelo menos uma tentativa real de envio. A janela de entrega expirou enquanto o destino estava inacessível. |
| `MESSAGE_EXPIRED` | Legado: expiração de **deliverUntil** ou comportamento de expiração antigo não coberto pelos códigos acima. |
| `DELIVERY_TIMEOUT` | Timeout de entrega, instância desconectada/fechada, instância não encontrada no provedor, ou outras falhas de entrega sem um código mais específico acima. |
| `NUMBER_NOT_FOUND_REMOVED` | A instância/número conectado foi removido no provedor. |
| `LINKED_WHATSAPP_INSTANCE_NOT_FOUND` | A chave de API estava vinculada a uma instância do WhatsApp que não existe mais (registro removido ou id inválido). |
| `WHATSAPP_NOT_EXIST` | O número de destino não está registrado no WhatsApp (resposta do provedor indicando número não registrado). |
| `META_TEMPLATE_NOT_APPROVED` | Template não aprovado para a Meta Cloud API. O template deve ser enviado e aprovado pela Meta antes de enviar mensagens por um número Meta. |
| `META_OUTSIDE_24H_WINDOW` | Janela de conversa de 24h fechada. Use um template aprovado para enviar mensagem a este número. |
| `META_NO_CONVERSATION` | Não existe conversa com este número. Inicie com um template aprovado. |
| `META_RATE_LIMITED` | Limite de envio da Meta atingido. Aguarde alguns minutos e tente novamente. |
| `META_TOKEN_INVALID` | O token da Meta é inválido ou expirou. Atualize o System Token nas configurações do número. |
| `META_SEND_FAILED` | Falha genérica de envio da Meta Cloud API. |
| `FREE_FORM_NOT_SUPPORTED` | Mensagens de formato livre não são suportadas para este provedor de WhatsApp. |
| `NUMBER_CONNECTION_LIMIT_REACHED` | Um único número de WhatsApp atingiu seu limite de 4 conexões. Remova uma conexão existente antes de adicionar uma nova. HTTP 409. |
| `META_WABA_NOT_ACCESSIBLE` | O ID da WABA (WhatsApp Business Account) associado a este número não existe ou o system token não tem permissão para acessá-lo. Retornado quando "Sincronizar com a Meta" falha com o código de erro 100 / subcódigo 33 da Meta. |
| `SEND_FAILED` | Todas as outras falhas (incluindo erros de envio HTTP ou do provedor, mensagens de diagnóstico genéricas e qualquer caso sem um código mais específico acima). |

<Note>
  Uma mensagem `CANCELED` (cancelada via [`DELETE /v1/messages/cancel`](/pt-BR/api/messages/cancel)) não é um erro de entrega e não carrega um desses códigos.
</Note>
